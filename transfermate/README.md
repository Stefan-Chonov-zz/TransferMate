# TransferMate Console App

### Dependencies
```
- PHP 8.0.3
- PostgreSQL 13.2
- Composer
```

### Installation

```
git clone https://github.com/Stefan-Chonov/TransferMate.git
cd TransferMate
cd transfermate
composer install
```

### Configs

```
- Environment configs are located in '.env' file in the path \TransferMate\transfermate\.env
```

### Run
```
cd \TransferMate\transfermate\
php run.php
```
